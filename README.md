## About The Project
<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

JAMA: Just Another Meteo App, is a simple meteo app showing the weather forecasts for the next 5 days.
App is localized in english and expresses data in metric system.

### Built With
* Swift 5.0
* Xcode 12.0
